# Альтернативный Следопыт (Kasoda) — v0.3.0 (база 1–20 + подкласс «Охотник»)

Модуль для Foundry VTT 13–14, система dnd5e 5.3.x.

## Установка
1. Распакуйте папку `sledopyt-kasoda` в `Data/modules/`.
2. Включите модуль в мире (Manage Modules). Должны быть включены: **dnd5e**, **Laaru•Compendium•DnD14** (`laaru-dnd5-hw`) и, желательно, RU-перевод (foundry-vtt-ru).
3. Откройте компендиум «Следопыт — Класс» → перетащите «Альтернативный Следопыт» на лист персонажа → пройдите advancement-флоу.

## Что внутри (v0.1)
- Класс 1–20: ХД d10, полузаклинатель (Мудрость), владения, ASI 4/8/12/16, выбор подкласса на 3-м, Боевой стиль на 2-м (пул Laaru).
- Scale Values: `hunt-die` (Кость охоты), `quarry-targets`, `quarry-sense`.
- 17 предметов-особенностей с дословным русским текстом из PDF.
- Заклинания от черт: «Язык природы» выдаёт «Разговор с животными/растениями» как always-prepared через advancement.
- Заряды (uses) с восстановлением по описанию: Первозданная осведомлённость (1+Мдр, ДО), Мастер исчезновения (БМ, ДО), Дикие чувства (1, КО/ДО), Охотничьи угодья (1, ДО).
- Шаблоны: Охотничьи угодья при использовании ставят measured template 30 фт (radius) на курсор.
- Active Effects: Вездеход (+скорости), Дикие чувства (слепое зрение), биом-адаптации и крит против Добычи — переключаемыми эффектами.

## Подклассы (батчами)
- **Охотник** (Батч 1): Поиск слабостей (3), Тактика охоты — выбор (3), Стратегия охоты — выбор (7), Магия охоты — заряды (11), Аватар охоты (15). Выбор-опции через ItemChoice; Стремительный охотник даёт +10 скорости эффектом.

## Пока НЕ входит
- Остальные 9 подклассов (следующие батчи).
- Трекинг ограниченных использований/Activity-броски (пас полировки).

Сборка источников: `python3 build.py` затем `node pack.mjs` (нужен @foundryvtt/foundryvtt-cli).
