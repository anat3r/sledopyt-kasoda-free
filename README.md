# Альтернативный Следопыт (Kasoda) — база 1–20 + подклассы

Модуль для Foundry VTT 13–14, система dnd5e 5.3.x.

## Установка

**Рекомендуется — по манифесту:** Foundry → Add-on Modules → Install Module → в поле Manifest URL вставьте:
```
https://github.com/anat3r/sledopyt-kasoda-dist/releases/latest/download/module.json
```

**Вручную (альтернатива):** распакуйте `module.zip` из релиза в `Data/modules/`.

Затем:
1. Включите модуль в мире (Manage Modules). Должны быть включены: **dnd5e**, **Laaru•Compendium•DnD14** (`laaru-dnd5-hw`) и, желательно, RU-перевод (foundry-vtt-ru).
2. Откройте компендиум «Следопыт — Класс» → перетащите «Альтернативный Следопыт» на лист персонажа → пройдите advancement-флоу.

> Раздача собранного модуля: **anat3r/sledopyt-kasoda-dist** (публичный). Исходники и сборка — в отдельном репозитории.

## Что внутри (v0.1)
- Класс 1–20: ХД d10, полузаклинатель (Мудрость), владения, ASI 4/8/12/16, выбор подкласса на 3-м, Боевой стиль на 2-м (пул Laaru).
- Scale Values: `hunt-die` (Кость охоты), `quarry-targets`, `quarry-sense`.
- 17 предметов-особенностей с дословным русским текстом из PDF.
- Заклинания от черт: «Язык природы» выдаёт «Разговор с животными/растениями» как always-prepared через advancement.
- Заряды (uses) с восстановлением по описанию: Первозданная осведомлённость (1+Мдр, ДО), Мастер исчезновения (БМ, ДО), Дикие чувства (1, КО/ДО), Охотничьи угодья (1, ДО).
- Шаблоны: Охотничьи угодья при использовании ставят measured template 30 фт (radius) на курсор.
- Active Effects: Вездеход (+скорости), Дикие чувства (слепое зрение), биом-адаптации и крит против Добычи — переключаемыми эффектами.

## Подклассы (батчами)
- **Охотник** (Батч 1): Поиск слабостей (3), Тактика охоты — выбор (3), Стратегия охоты — выбор (7), Магия охоты — заряды (11), Аватар охоты (15).
- **Сумрачный охотник** (Батч 2): Угроза из засады + Теневой взор (3), Родство с темнотой (7), Источник ужаса (11), Аватар ужаса (15); спелл-лист 3/5/9/13/17.
- **Убийца чудовищ** (Батч 2): Универсальный охотник (3), Сверхъест. осведомлённость + Тщательная подготовка (7), Смещение чувств (11), Стайный охотник (15); спелл-лист 3/5/9/13/17.
