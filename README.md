# Альтернативный Следопыт (Kasoda) — бесплатная версия

Бесплатная (демонстрационная) редакция альтернативного класса **Следопыт** от Kasoda для **Foundry VTT 13–14**, система **dnd5e 5.3.x**. Основана на бесплатной версии PDF: полный класс 1–20 и два подкласса — **Охотник** и **Повелитель зверей**.

> Полная версия со всеми 10 подклассами распространяется отдельно автором оригинала (Kasoda).

## Установка

**Рекомендуется — по манифесту:** Foundry → Add-on Modules → Install Module → в поле **Manifest URL** вставьте:
```
https://github.com/anat3r/sledopyt-kasoda-free/releases/latest/download/module.json
```

**Вручную (альтернатива):** распакуйте `module.zip` из последнего релиза в `Data/modules/`.

Затем:
1. Включите модуль в мире (Manage Modules). Желательно также включить **Laaru•Compendium•DnD14** (`laaru-dnd5-hw`) — из него по UUID линкуются заклинания always-prepared и боевые стили; без него класс работает, но эти ссылки не разрешатся.
2. Откройте компендиум «Следопыт — Класс» → перетащите «Альтернативный Следопыт» на лист персонажа → пройдите advancement-флоу (выбор навыков, боевого стиля на 2-м, подкласса на 3-м и т.д.).

## Что внутри (v0.2)

**Базовый класс 1–20** (идентичен полной версии):
- ХД d10, полузаклинатель от Мудрости (ячейки с 1-го уровня), владения, ASI 4/8/12/16, выбор подкласса на 3-м, Боевой стиль на 2-м (пул из Laaru, включая «Воин-друид»).
- Scale Values: `hunt-die` (Кость охоты d4→d10), `quarry-targets` (Цели Охоты), `quarry-sense` (Чутьё Добычи).
- 17 предметов-особенностей с дословным русским текстом из PDF.
- Always-prepared заклинания от черт («Язык природы» → «Разговор с животными/растениями»).
- Заряды (uses) с восстановлением по описанию: Первозданная осведомлённость, Мастер исчезновения, Дикие чувства, Охотничьи угодья.
- Шаблон: Охотничьи угодья ставят measured template 30 фт (radius) на курсор.
- Active Effects: Вездеход (+скорости), Дикие чувства (слепое зрение), биом-адаптации и крит против Добычи — переключаемыми (toggle) эффектами.

**Подклассы (2):**
- **Охотник:** Поиск слабостей (3), Тактика охоты — выбор (3), Стратегия охоты — выбор (7), Магия охоты — заряды (11), Аватар охоты (15).
- **Повелитель зверей:** Спутник следопыта (3) — владение «Уход за животными», always-prepared «Животные чувства»/«Разговор с животными» и **активность призыва Зверя-спутника**; Зачарованный спутник (7) — **выбор черт** (3/4/5 на 7/11/15); Могучий спутник (11); Неразрывная связь (15).

> **Зверь-спутник** реализован как dnd5e-активность призыва в режиме «по уровню опасности» (как Conjure Beast у Laaru): кнопка **«Призвать Зверя-спутника»** ставит токен **любого зверя** с ПО ≤ половины вашего уровня (из доступных бестиариев, напр. Laaru), автоматически применяя ваш Бонус мастерства к КД/урону и Мудрость к атакам/спасброскам/навыкам. Никакой статблок в модуль не зашит.
>
> **Черты Зверя-спутника** (Звериная мощь, Адаптируемое передвижение, Зверь-преследователь, Природный камуфляж, и с 11-го уровня — Неуловимый/Стайный/Кровожадный) — отдельные особенности с выбором (ItemChoice), переключаемыми эффектами (скорости, иммунитеты — перетаскиваются на токен Зверя) и кнопками-бросками урона `[[/damage]]` прямо в тексте.

## Не входит (бесплатная версия)
- Остальные 8 подклассов (только в полной версии).
- Полная автоматизация ситуативных эффектов — там, где это поле для отыгрыша мастер↔игрок, оставлен текст или переключаемый эффект.

## Сборка из исходников
```bash
npm ci            # зависимости (@foundryvtt/foundryvtt-cli)
npm run validate  # проверка целостности (id/_key/внутренние ссылки)
npm run pack      # src/ -> packs/ (LevelDB)
npm run generate  # (опц.) перегенерировать src/ из tools/*.py
```
Источник истины — `src/**/*.json`. `packs/` собираются из них и в git не хранятся.
